SpeShu.AI работает как посредник между пользователями и разными AI-провайдерами. Когда вы отправляете запрос, он передаётся выбранному провайдеру. В некоторых случаях запрос может проходить через цепочку провайдеров — это зависит от конкретной модели и маршрута обработки.

## Что SpeShu.AI не хранит

**Запросы.** Пользовательские запросы к AI-моделям не сохраняются по умолчанию.

**Ответы.** Ответы от AI-систем не логируются по умолчанию.

**Личные данные.** SpeShu.AI не собирает личную информацию из переписки и не использует содержимое запросов для собственных целей.

## Что SpeShu.AI сохраняет

**Статистику использования.** Мы сохраняем технические метаданные: количество токенов, время ответа, используемую модель и другую информацию, необходимую для расчётов, аналитики и стабильной работы сервиса.

**Мультимедийные файлы.** Результаты AI-генераций, например изображения или другие медиафайлы, хранятся 7 дней, чтобы обеспечить быстрый доступ к ним.

## Политика провайдеров

SpeShu.AI работает с разными AI-провайдерами. Их политики обработки данных отличаются, поэтому мы делим провайдеров на три группы: безопасные, с временным хранением и потенциально небезопасные.

### Безопасные провайдеры

Эти провайдеры не хранят данные и не используют их для обучения моделей.

<Note>
    Не хранят данные, не используют для обучения моделей.
</Note>

| Провайдер | Особенности |
| --- | --- |
| Azure | Enterprise-grade безопасность |
| Groq | Высокая скорость и no-logging |
| Amazon Bedrock | Инфраструктура AWS |
| DeepInfra | Оптимизированный инференс |
| Fireworks | Быстрый инференс |
| Together | Open-source модели |
| Cerebras | Специализированное железо |
| SambaNova | Enterprise-решения |
| Featherless | Приватность по умолчанию |

### Провайдеры с временным хранением

Эти провайдеры могут временно хранить данные и удаляют их в срок от 1 до 30 дней.

<Warning>
    Удаляют данные через 1-30 дней.
</Warning>

| Провайдер | Срок хранения |
| --- | --- |
| OpenAI | До 30 дней |
| Anthropic | До 30 дней |
| Cohere | До 30 дней |
| Google Vertex | До 30 дней |
| Mistral | До 30 дней |
| xAI | До 30 дней |
| Inflection | До 30 дней |

### Потенциально небезопасные провайдеры

Эти провайдеры могут использовать данные для обучения моделей или не дают достаточных гарантий по обработке информации.

<Error>
    Могут использовать данные для обучения моделей.
</Error>

| Провайдер | Риск |
| --- | --- |
| DeepSeek | Возможно использование данных для обучения |
| OpenInference | Нет достаточных гарантий |

<ImportantNote>
При работе с чувствительными, коммерческими или персональными данными рекомендуем выбирать провайдеров из категории безопасных и заранее проверять политику конкретной модели.
</ImportantNote>
