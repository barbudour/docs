Здесь вы узнаете о базовых шагах: регистрация в SpeShu.AI, создание организации, получение API-ключа и пополнение счёта.

## Регистрация и организация

Сначала нужно создать аккаунт. Перейдите на сайт [speshu.ai](https://speshu.ai) и пройдите регистрацию. После входа можно оформить организацию внутри кабинета.

## API-ключ

Чтобы работать с API, потребуется ключ.

<Steps>
  <Step title="Откройте личный кабинет">
    [speshu.ai/profile](https://speshu.ai/profile)
  </Step>

  <Step title="Перейдите в раздел API-ключи">
    Создайте новый ключ
  </Step>

  <Step title="Задайте имя ключу">
    Например для продакшена или разработки
  </Step>

  <Step title="Скопируйте и сохраните">
    Значение ключа не показывается повторно — сохраните его сразу
  </Step>
</Steps>

<Note>
  Ключ лучше хранить отдельно и не публиковать. Если есть риск компрометации — удалите его и выпустите новый.
</Note>

## Баланс

В SpeShu.AI используется оплата по факту использования — деньги списываются за запросы. Перед началом работы нужно пополнить счёт.

<AccordionGroup>
  <Accordion title="Банковская карта" icon="credit-card">
    Поддерживаются Visa, Mastercard и МИР. Зачисление происходит сразу, минимальный платёж — 100 ₽.

    Чтобы оплатить:
    * откройте раздел баланса в профиле
    * нажмите «Пополнить»
    * укажите сумму и подтвердите оплату
  </Accordion>

  <Accordion title="СБП" icon="mobile">
    Оплата через банковское приложение по QR-коду. Деньги поступают сразу.

    Порядок действий:
    * выберите СБП как способ оплаты
    * отсканируйте код
    * подтвердите платёж
  </Accordion>

  <Accordion title="Счёт для юрлиц" icon="building">
    Подходит для организаций. Зачисление занимает 1–2 рабочих дня после оплаты.

    Что нужно сделать:
    * выбрать оплату по счёту
    * заполнить реквизиты
    * скачать документ и оплатить его
  </Accordion>
</AccordionGroup>

## Уведомления о низком балансе

Чтобы не остаться без средств в момент запроса, включите оповещения:

1. Перейдите в [Настройки](https://speshu.ai/profile)
2. В разделе «Уведомления» активируйте уведомления о низком балансе
3. Задайте порог, например 500 ₽

При достижении порога придёт письмо на email.

## Проверка баланса через API

Баланс можно получать программно:

<CodeGroup>
  ```typescript TypeScript theme={null}
  const response = await fetch('https://speshu.ai/api/v1/balance', {
    headers: {
      'Authorization': 'Bearer <SPESHU_AI_API_KEY>'
    }
  });

  const data = await response.json();
  console.log(`Баланс: ${data.amount} руб.`);
  console.log(`Потрачено: ${data.spentAmount} руб.`);
  ```

  ```python Python theme={null}
  import requests

  response = requests.get(
      'https://speshu.ai/api/v1/balance',
      headers={'Authorization': 'Bearer <SPESHU_AI_API_KEY>'}
  )

  data = response.json()
  print(f"Баланс: {data['amount']} руб.")
  print(f"Потрачено: {data['spentAmount']} руб.")
  ```

  ```bash cURL theme={null}
  curl "https://speshu.ai/api/v1/balance" \
    -H "Authorization: Bearer <SPESHU_AI_API_KEY>"
  ```
</CodeGroup>

### Пример ответа

```json theme={null}
{
  "amount": "1250.50"
}
```

| Поле     | Тип    | Описание                |
| -------- | ------ | ----------------------- |
| `amount` | string | Текущий баланс в рублях |
