Dify — платформа с открытым исходным кодом для создания AI-приложений: чат-ботов, агентов и автоматизированных Workflow. Поддерживает подключение любых моделей через OpenAI-совместимый API, что позволяет использовать весь каталог Speshu — Claude, Gemini, DeepSeek и другие — прямо из интерфейса Dify.

## Требования

* Аккаунт на [cloud.dify.ai](https://cloud.dify.ai) (или self-hosted инстанс)
* API-ключ Speshu — получить на [speshu.ai/profile](https://speshu.ai/profile)

## Подключение Speshu

### 1. Регистрация и главный раздел

Зайдите на [cloud.dify.ai](https://cloud.dify.ai), создайте аккаунт или войдите в существующий. После входа вы попадёте на главный экран платформы.

<img src="https://mintcdn.com/polza/ilNnQ-isgxx3Mhnt/images/dify/01-main.png?fit=max&auto=format&n=ilNnQ-isgxx3Mhnt&q=85&s=98014b970a309cc573963c1912c291fb" alt="Главный экран Dify после входа" width="1677" height="883" data-path="images/dify/01-main.png" />

### 2. Поставщики моделей

Откройте **Настройки** → **Поставщики моделей**. В списке провайдеров найдите и выберите **OpenAI-API-compatible**.

<img src="https://mintcdn.com/polza/ilNnQ-isgxx3Mhnt/images/dify/02-providers.png?fit=max&auto=format&n=ilNnQ-isgxx3Mhnt&q=85&s=f44ddd1dcfc965c2129efdc7a8f54679" alt="Раздел Поставщики моделей, выбор OpenAI-API-compatible" width="1675" height="885" data-path="images/dify/02-providers.png" />

### 3. Добавление модели

Нажмите **Добавить модель** и заполните поля:

| Поле             | Значение                                                    |
| ---------------- | ----------------------------------------------------------- |
| Model Name       | Любое имя, например `claude-sonnet-4.6`                     |
| API Key          | Ваш ключ с [speshu.ai/profile](https://speshu.ai/profile) |
| API endpoint URL | `https://speshu.ai/api/v1`                                   |

<Note>
    Идентификаторы доступных моделей — на странице [speshu.ai/models](https://speshu.ai/models). Вы можете добавить несколько моделей, повторив этот шаг.
</Note>

<img src="https://mintcdn.com/polza/ilNnQ-isgxx3Mhnt/images/dify/03-add-model.png?fit=max&auto=format&n=ilNnQ-isgxx3Mhnt&q=85&s=710d52c3256e13ae031d6f9828a18157" alt="Форма добавления модели с заполненными полями" width="1672" height="882" data-path="images/dify/03-add-model.png" />

### 4. Создание ИИ Агента

Вернитесь на главную и создайте новое приложение, например, **Чатфлоу с ИИ агентом**.

<img src="https://mintcdn.com/polza/ilNnQ-isgxx3Mhnt/images/dify/04-create-agent.png?fit=max&auto=format&n=ilNnQ-isgxx3Mhnt&q=85&s=d9fd120549c404fac8b5fd06c08177fc" alt="Создание нового агента в Dify" width="1676" height="888" data-path="images/dify/04-create-agent.png" />

### 5. Workflow

Добавьте нужные блоки — LLM, инструменты, условия — и укажите модель Speshu в блоке LLM. Настройте параметры модели.

<img src="https://mintcdn.com/polza/ilNnQ-isgxx3Mhnt/images/dify/05-workflow.png?fit=max&auto=format&n=ilNnQ-isgxx3Mhnt&q=85&s=8016b5a477bdebe2d867e69ab53e703c" alt="Редактор Workflow с блоком LLM на базе Speshu" width="1652" height="880" data-path="images/dify/05-workflow.png" />

### 6. Проверка подключения

Откройте **Предпросмотр** и отправьте тестовый запрос. Если в ответе появляется сообщение модели — подключение работает.

<img src="https://mintcdn.com/polza/ilNnQ-isgxx3Mhnt/images/dify/06-preview.png?fit=max&auto=format&n=ilNnQ-isgxx3Mhnt&q=85&s=d81df8134b50afe39e46fb561c84f2e1" alt="Предпросмотр с успешным ответом модели" width="1677" height="870" data-path="images/dify/06-preview.png" />

## Дальнейшая настройка

Dify предоставляет широкие возможности для кастомизации: системные промпты, инструменты и API-вызовы, RAG-базы знаний, многоэтапные Workflow, публикация в виде веб-приложения или API. Настраивайте сценарий под свои задачи — ограничений по типу использования нет.

## Решение проблем

<AccordionGroup>
    <Accordion title="Модель не отвечает / ошибка при тестировании">
        Проверьте правильность API-ключа и URL (`https://speshu.ai/api/v1`). Убедитесь, что в поле **Model Name** указан точный идентификатор модели из [speshu.ai/models](https://speshu.ai/models).
    </Accordion>

    <Accordion title="Модель не появляется в списке при создании агента">
        После добавления модели обновите страницу. Если модель всё равно не отображается — убедитесь, что сохранили настройки поставщика.
    </Accordion>

    <Accordion title="Ошибка 401 Unauthorized">
        API-ключ недействителен или истёк. Сгенерируйте новый ключ на [speshu.ai/profile](https://speshu.ai/profile) и обновите его в настройках поставщика.
    </Accordion>
</AccordionGroup>
