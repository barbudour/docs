Nano Banana Pro — улучшенная версия с поддержкой генерации изображений до 4K разрешения, точным рендерингом текста и до 8 референс-изображений.

> Полная документация API: [Генерация медиа](/api-reference/media/create)

## Обзор

| Характеристика       | Nano Banana                     | Nano Banana Pro                     |
| -------------------- | ------------------------------- | ----------------------------------- |
| ID модели            | `google/gemini-2.5-flash-image` | `google/gemini-3-pro-image-preview` |
| Max prompt           | 20 000 символов                 | 20 000 символов                     |
| Разрешения           | -                               | 1K, 2K, 4K                          |
| Aspect ratio         | 10 вариантов                    | 11 вариантов + `auto`               |
| Референс-изображения | до 8                            | до 8                                |

## Преимущества Pro версии

- **Высокое разрешение**: Генерация изображений до 4K разрешения
- **Точный рендеринг текста**: Улучшенная генерация текста на изображениях
- **Авто-соотношение**: Автоматический выбор оптимального aspect ratio
- **Детализация**: Более точная проработка мелких деталей

## Сценарии использования

- **Профессиональная фотография**: Коммерческие съёмки и каталоги
- **Печатная продукция**: Баннеры, постеры, полиграфия высокого качества
- **Детальные иллюстрации**: Сложные композиции с множеством элементов
- **Брендинг**: Визуальные материалы для брендов

## Параметры

| Параметр           | Тип    | Обязательный | Описание                                                                                            |
| ------------------ | ------ | ------------ | --------------------------------------------------------------------------------------------------- |
| `model`            | string | Да           | ID модели: `google/gemini-3-pro-image-preview`                                                      |
| `prompt`           | string | Да           | Текстовое описание желаемого изображения (до 20 000 символов)                                       |
| `aspect_ratio`     | string | Нет          | Соотношение сторон: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9`, `auto` |
| `image_resolution` | string | Нет          | Разрешение: `1K`, `2K`, `4K`                                                                        |
| `output_format`    | string | Нет          | Формат вывода: `png`, `jpeg`                                                                        |
| `images`           | array  | Нет          | Массив референс-изображений для редактирования (до 8 шт.)                                           |

## Пример запроса (cURL)

```bash
curl -X POST "https://polza.ai/api/v1/media" \
  -H "Authorization: Bearer <POLZA_AI_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "google/gemini-3-pro-image-preview",
    "input": {
      "prompt": "Описание изображения",
      "aspect_ratio": "1:1",
      "image_resolution": "2K",
      "images": []
    },
    "async": true
  }'
```

## Сравнение с базовой версией

| Функция                 | Nano Banana | Nano Banana Pro |
| ----------------------- | ----------- | --------------- |
| Максимальное разрешение | Стандартное | До 4K           |
| Точность текста         | Базовая     | Улучшенная      |
| Авто aspect ratio       | Нет         | Да              |
| Детализация             | Стандартная | Повышенная      |

## Советы и лучшие практики

- **Используйте 4K для печати**: Для печатной продукции выбирайте `image_resolution: "4K"` для максимального качества.
- **Детализируйте освещение**: Pro версия отлично обрабатывает сложные световые схемы — описывайте их подробно.
- **Используйте auto для соотношения**: Параметр `aspect_ratio: "auto"` позволит модели выбрать оптимальное соотношение для вашего контента.
