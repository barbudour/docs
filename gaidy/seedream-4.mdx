Seedream 4 — модель генерации изображений от ByteDance с уникальным художественным стилем и поддержкой до 4K разрешения.

<Note>
  Полная документация API: [Генерация медиа](/api-reference/media/create)
</Note>

## Обзор

| Характеристика       | Значение                                  |
| -------------------- | ----------------------------------------- |
| Тип                  | Генерация изображений                     |
| ID модели            | `bytedance/seedream-4`                    |
| Max prompt           | 5 000 символов                            |
| Разрешения           | 1K, 2K, 4K                                |
| Соотношения сторон   | 1:1, 4:3, 3:4, 3:2, 2:3, 16:9, 9:16, 21:9 |
| Референс-изображения | до 10                                     |

## Отличительные особенности

- **Художественный стиль**: Уникальная стилизация изображений
- **Контроль генерации**: Точная настройка через seed для воспроизводимости результатов
- **Высокое разрешение**: Генерация до 4K разрешения

## Сценарии использования

- **Концепт-арт**: Создание концептов для игр и фильмов
- **Обложки**: Дизайн обложек книг и альбомов
- **Арт-проекты**: Художественные и творческие проекты
- **Стилизация**: Создание изображений в определённом стиле

## Параметры

| Параметр           | Тип    | Обязательный | Описание                                                                      |
| ------------------ | ------ | ------------ | ----------------------------------------------------------------------------- |
| `model`            | string | Да           | ID модели: `bytedance/seedream-4`                                             |
| `prompt`           | string | Да           | Текстовое описание желаемого изображения (до 5 000 символов)                  |
| `aspect_ratio`     | string | Нет          | Соотношение сторон: `1:1`, `4:3`, `3:4`, `3:2`, `2:3`, `16:9`, `9:16`, `21:9` |
| `image_resolution` | string | Нет          | Разрешение: `1K`, `2K`, `4K`                                                  |
| `seed`             | number | Нет          | Seed для воспроизводимости результатов (1–4294967295)                         |
| `images`           | array  | Нет          | Массив референс-изображений для редактирования (до 10 шт.)                    |

## Пример запроса (cURL)

```bash theme={null}
curl -X POST "https://speshu.ai/api/v1/media" \
  -H "Authorization: Bearer <SPESHU_AI_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "bytedance/seedream-v4",
    "input": {
      "prompt": "Описание изображения",
      "aspect_ratio": "16:9",
      "image_resolution": "2K",
      "seed": 12345,
      "images": []
    },
    "async": true
  }'
```

## Советы и лучшие практики

**Указывайте стиль явно**
Seedream 4 хорошо работает со стилистическими указаниями: «в стиле аниме», «импрессионизм», «киберпанк».

**Используйте seed для вариаций**
Зафиксируйте `seed` для воспроизводимости или меняйте его для получения вариаций удачной генерации.
