Nano Banana — модель генерации изображений от Google с естественной обработкой языка и поддержкой длинных промптов до 20 000 символов.

<Note>
  Полная документация API: [Генерация медиа](/api-reference/media/create)
</Note>

## Обзор

| Характеристика       | Значение                                            |
| -------------------- | --------------------------------------------------- |
| Тип                  | Генерация изображений                               |
| ID модели            | `google/gemini-2.5-flash-image`                     |
| Max prompt           | 20 000 символов                                     |
| Соотношения сторон   | 1:1, 9:16, 16:9, 3:4, 4:3, 3:2, 2:3, 5:4, 4:5, 21:9 |
| Форматы вывода       | png, jpeg                                           |
| Референс-изображения | до 8                                                |

## Параметры

| Параметр        | Тип    | Обязательный | Описание                                                                                    |
| --------------- | ------ | ------------ | ------------------------------------------------------------------------------------------- |
| `model`         | string | Да           | ID модели: `google/gemini-2.5-flash-image`                                                  |
| `prompt`        | string | Да           | Текстовое описание желаемого изображения (до 20 000 символов)                               |
| `n`             | number | Нет          | Количество изображений (по умолчанию 1, максимум 8)                                         |
| `aspect_ratio`  | string | Нет          | Соотношение сторон: `1:1`, `9:16`, `16:9`, `3:4`, `4:3`, `3:2`, `2:3`, `5:4`, `4:5`, `21:9` |
| `output_format` | string | Нет          | Формат вывода: `png`, `jpeg`                                                                |
| `images`        | array  | Нет          | Массив референс-изображений для редактирования (до 8 шт.)                                   |

## Пример запроса (cURL)

```bash theme={null}
curl -X POST "https://speshu.ai/api/v1/media" \
  -H "Authorization: Bearer <SPESHU_AI_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "google/gemini-2.5-flash-image",
    "input": {
      "prompt": "Описание изображения",
      "aspect_ratio": "16:9",
      "images": []
    },
    "async": true
  }'
```

## Советы и лучшие практики

**Пишите детальные промпты**
Чем подробнее описание, тем точнее результат. Указывайте стиль, освещение, композицию, цветовую гамму.

**Используйте длинные промпты**
Nano Banana отлично работает с промптами до 20 000 символов — описывайте сцену максимально подробно.

**Экспериментируйте с соотношениями сторон**
Выбирайте aspect_ratio в зависимости от цели: 1:1 для аватаров, 16:9 для баннеров, 9:16 для сторис.
