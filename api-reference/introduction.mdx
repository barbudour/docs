Speshu предоставляет REST API, совместимый со стандартом OpenAI, для доступа к сотням AI-моделей.

## Базовый URL

```
https://speshu.ai/api
```

## Аутентификация

Все запросы требуют API-ключ в заголовке `Authorization`:

```bash theme={null}
Authorization: Bearer YOUR_API_KEY
```

<Note>
    Получите API-ключ в [консоли](https://speshu.ai/dashboard/api-keys).
</Note>

## Совместимость с OpenAI

Speshu полностью совместим с OpenAI SDK. Просто измените `base_url`:

<CodeGroup>
    ```python Python theme={null}
    from openai import OpenAI

    client = OpenAI(
    base_url="https://speshu.ai/api/v1",
    api_key="<SPESHU_AI_API_KEY>"
    )
    ```

    ```typescript TypeScript theme={null}
    import OpenAI from 'openai';

    const client = new OpenAI({
        baseURL: 'https://speshu.ai/api/v1',
        apiKey: '<SPESHU_AI_API_KEY>'
    });
    ```
</CodeGroup>

## Коды ответов

| Код | Описание                                  |
| --- | ----------------------------------------- |
| 200 | Успешный запрос                           |
| 201 | Задача создана (для асинхронных операций) |
| 400 | Неверный запрос                           |
| 401 | Ошибка аутентификации                     |
| 402 | Недостаточно средств                      |
| 403 | Доступ запрещён                           |
| 404 | Ресурс не найден                          |
| 408 | Таймаут запроса                           |
| 429 | Превышен лимит запросов                   |
| 500 | Ошибка сервера                            |
| 502 | Провайдер недоступен                      |
| 503 | Нет доступных провайдеров                 |

## Формат ошибок

```json theme={null}
{
  "error": {
    "code": "NOT_FOUND",
    "message": "Ресурс не найден"
  }
}
```
