---
title: "Удалить файл"
description: "Удаление файла из хранилища"
openapi: DELETE /v1/storage/files/{id}
playground: interactive
icon: "trash"
---

## Пример

```bash
curl -X DELETE "https://speshu.ai/api/v1/storage/files/file_abc123" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Ответ (200)

```json
{
  "success": true
}
```

Удаление файла необратимо. Файл будет удалён из хранилища и CDN.
