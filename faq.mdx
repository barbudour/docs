## Начало работы

<AccordionGroup>
  <Accordion title="Что такое SpeShu.AI?">
      SpeShu.AI — это удобный сервис с единым API для доступа к популярным LLM-моделям. Мы объединяем модели OpenAI, Anthropic, Google и десятков других провайдеров в одном интерфейсе.
  </Accordion>

  <Accordion title="Как начать пользоваться SpeShu.AI?">
    1. Создайте аккаунт на [speshu.ai](https://speshu.ai)
    2. Пополните баланс в личном кабинете или проведите оплату по счету.
    3. Создайте API-ключ.
    4. Подключитесь к нашему API через любой OpenAI-совместимый SDK.
  </Accordion>

  <Accordion title="Как устроено ценообразование?">
      Стоимость указывается за миллион токенов и отдельно рассчитывается для входящих и исходящих токенов. Для некоторых моделей дополнительно может тарифицироваться количество запросов, генерация изображений или токены рассуждений. Историю использования можно посмотреть в Консоли.
  </Accordion>
</AccordionGroup>

## Модели и провайдеры

<AccordionGroup>
  <Accordion title="Какие модели доступны на SpeShu.AI?">
      Полный список из +300 моделей доступен через API моделей. Мы регулярно добавляем новые модели и провайдеров.
  </Accordion>

  <Accordion title="Какие SDK поддерживает SpeShu.AI?">
      SpeShu.AI полностью совместим с OpenAI API. Поэтому с сервисом работают любые SDK, поддерживающие OpenAI, включая официальные библиотеки для Python, Node.js, Go и других языков.
  </Accordion>
</AccordionGroup>

  <Accordion title="Поддерживает ли API изображения и другие файлы?">
      Да. Для мультимодальных моделей поддерживаются текст, изображения и PDF — в формате URL или base64.
  </Accordion>

  <Accordion title="Как работает потоковая передача?">
      Потоковая передача работает через Server-Sent Events (SSE). Чтобы получать ответ частями, добавьте в запрос параметр `stream: true`.
  </Accordion>
</AccordionGroup>

## Конфиденциальность и данные

<AccordionGroup>
  <Accordion title="Какие данные сохраняет SpeShu.AI?">
      По умолчанию мы сохраняем только базовые метаданные запросов: время, используемую модель и количество токенов. Содержимое запросов и ответов не сохраняется. При создании или редактировании API-ключа можно дополнительно включить логирование. Срок хранения логов — 30 дней.
  </Accordion>

  <Accordion title="Как SpeShu.AI передаёт данные провайдерам?">
      Мы передаём запросы провайдерам и работаем с ними так, чтобы данные не использовались для обучения моделей. Подробнее об этом можно прочитать в разделе Конфиденциальность.
  </Accordion>
</AccordionGroup>

## Оплата и баланс

<AccordionGroup>
  <Accordion title="Как устроена система оплаты?">
      Баланс пополняется в рублях. Доступно ручное пополнение или автоматическое — когда баланс опускается ниже заданного порога.
  </Accordion>

  <Accordion title="Какие способы оплаты доступны?">
      Мы принимаем банковские карты, оплату через СБП и оплату на расчётный счёт для бизнеса.
  </Accordion>

  <Accordion title="Какова политика возврата?">
      Возврат неиспользованных средств возможен в течение 24 часов после оплаты. Для оформления возврата обратитесь в [службу поддержки](/support).
  </Accordion>

  <Accordion title="Как отслеживать расходы?">
      Актуальную информацию о расходах можно посмотреть на странице [speshu.ai/profile](https://speshu.ai/profile) или через API баланса.
  </Accordion>

  <Accordion title="Есть ли скидки за объём?">
      Стандартных скидок за объём нет, но для крупных клиентов возможны индивидуальные условия. Свяжитесь с нами, чтобы обсудить формат сотрудничества.
  </Accordion>
</AccordionGroup>
